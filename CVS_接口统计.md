#### 认证控制器

1. 登录
2. 注册
3. 登出
4. 重置密码
5. 获取邮箱/或者手机验证码
6. 验证验证码

#### 活动控制器

1. 创建活动（可以直接发布或者保存到草稿）
2. 根据id查看单个活动详细信息
3. 查看活动列表
   1. 管理员端：
      + 分页查询所有活动列表
      + 通过标题/创建者名字查询活动列表，优先按照时间排序，可筛选发布状态（发布/草稿/取消）
   2. 教师端：
      + 分页查询当前登录教师的活动列表
      + 通过标题查询当前登录教师的活动列表，优先按照时间顺序，可筛选发布状态
   3. 学生端：
      + 分页查询所有活动列表（优先展示还可以报名的活动）
      + 通过标题/活动类型筛选活动列表

4. 取消/删除/发布的活动
5. 编辑活动

#### 活动报名控制器

1. 报名活动（student）
2. 查询报名详情
3. 查询报名列表
   1. 学生：可直接分页查全部，或者按照状态筛选。
   2. 老师：分页查全部；按照活动名称/学生名称（模糊查询）/审核状态筛选
4. 通过/批量通过报名
5. 拒绝/批量拒绝报名
6. 签到/签退（老师操作），签退后自动创建服务记录，并且根据评分发放积分

#### 服务记录控制器

1. 签退后自动创建服务记录；签退界面添加服务描述、服务评价、评分这些信息
2. 老师编辑服务记录
3. 分页查询服务记录（老师只能查询自己创建活动的，学生只能查询自己参与的）；按照活动名称/用户名查询

#### 用户控制器

1. 查看个人资料
2. 修改个人资料
3. 查询用户列表（admin）

#### 统计控制器

1. 仪表盘
   1. admin：老师数量、学生数量、总活动数、近期活动数
   2. teacher：我的活动、参与学生、待审核报名、服务记录审批
   3. student：我的报名、服务总时长、积分总数、服务记录
